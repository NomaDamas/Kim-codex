Submission bundle

해커톤 제출자가 바로 설명할 수 있는 산출물입니다.

web demo urlready
로컬 실행: npm start 후 http://localhost:4173 로그인, API key, secret이 필요 없습니다.
static html bundleready
npm run bundle이 dist/index.html을 생성합니다. 정적 HTML 파일 하나만 열어도 동일한 데모가 실행됩니다.
safety scopeimportant
실제 LinkedIn/Threads DM 전송은 하지 않습니다. MVP는 computer-use 컨셉, 병렬성, 브라우저 UI, 메시지 초안, SC2 루프를 증명합니다.
Run full demo를 누르면 제출용 리포트가 여기에 생성됩니다.

MVP boundary: fictional demo profiles, consent-safe draft-only outreach, no credential storage, no network dependency.
